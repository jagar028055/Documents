# 2026年FP1級試験対策

## 第1問：ライフプランニングと資金計画
### 国民年金の届出・納付
**資格取得の届出：14日以内**

**保険料の納付方法：**

| 区分 | 対象 | 期限 | メリット・備考 |
| :--- | :--- | :--- | :--- |
| **前納** | 今後の保険料を先払い | 最長2年分（口座振替・クレカ） | 割引あり |
| **通常納付** | 未納の保険料 | 納付期限から**2年以内** | 時効前なら納付可 |
| **後納**（特例） | 時効（2年）を過ぎた未納分 | **2018年9月末で終了** | 現在利用不可 |
| **追納** | 免除・猶予・学生特例期間 | **免除等を受けた月から10年以内**（承認を受けた月の前10年以内） | 年金額増加、社会保険料控除 |

**重要：**
- **2年以内の未納** → 通常通り納付可能（時効前）
- **2年を超えた未納** → 後納制度が終了したため、現在は納付不可
- **免除・猶予を受けた期間** → 追納（10年以内）で年金額を増やせる
- 追納は所得控除の対象になるため、節税効果あり

### 健康保険
**任意継続被保険者：**
- 申請期限：退職日の翌日から**20日以内**
- 届出先：全国健康保険協会の都道府県支部
- 加入期間：最長**2年間**

**高額療養費：**
- **世帯合算**：70歳未満は、1人あたり**2万1,000円以上**の自己負担があれば合算可
- **多数回該当**：過去12ヶ月で**4回目**から自己負担限度額が軽減
- **考え方**：窓口で支払った自己負担（原則3割等）のうち、所得区分ごとの**自己負担限度額**を超えた分が高額療養費として支給（または限度額適用認定証等により窓口負担が限度額までになる）
- **現物給付**：治療行為そのものを提供すること（金銭ではなく医療サービスを直接提供）

### 年金額の計算
- 831,700円…令和7年度における老齢基礎年金の満額（年額）
    - 831,700×454/480月
    - 繰上げする場合は、繰上げ月数×0.4％
- 特別支給の老齢厚生年金（定額部分）
    - 定額単価（R7年度 1,734円）× 乗数 × 加入月数（上限480）
- 老齢厚生年金（報酬比例部分） ＝ ① ＋ ②
    - ① H15.3以前：平均標準報酬月額 × 7.125/1000 × 加入月数
    - ② H15.4以降：平均標準報酬額 × 5.481/1000 × 加入月数
    - ※物価・賃金スライド（再評価率）を乗じて現在価値に直す
- 経過的加算（基礎年金と厚生年金の差額加算）
    - 定額部分の額 －（老齢基礎年金相当額 ＝ 831,700円 × 20歳以上60歳未満の加入月数 / 480）
- 加給年金（厚生年金加入20年以上＋生計維持家族）
    - 配偶者：**239,300円** ＋ 特別加算（受給者の生年月日に応じ最大**173,300円**）
    - 子：1人目・2人目 各**239,300円**、3人目以降 各**79,800円**
    - ※R7年度価格
- 高年齢雇用継続給付（雇用保険）
    - **高年齢雇用継続基本給付金**：60歳以降の賃金が一定以上低下した場合に支給（支給率は賃金低下率に応じて逓減）
        - 60歳到達日が**2025/4/1以後**：支給率の上限 **10%**
        - 60歳到達日が**2025/3/31以前**：支給率の上限 **15%**
    - 老齢厚生年金との併給調整（支給停止）
        - 上限：標準報酬月額の **4%**（60歳到達日が2025/4/1以後）／ **6%**（2025/3/31以前）

## 第2問：金融資産運用
### 財務分析
- 総資産経常利益率
    - 売上高経常利益率
    - 総資産回転率
- インタレスト・カバレッジ・レシオ＝事業利益/支払利息
    - 事業利益（例）＝営業利益＋受取利息＋受取配当金（＝金融収益）
- サスティナブル成長率＝ROE×（1－配当性向）
    - ROE＝当期純利益÷自己資本（純資産-非支配株主持分）
- 使用総資本事業利益率＝事業利益÷使用総資本（投下資本）
    - 使用総資本（例）＝自己資本＋有利子負債
    - 事業利益（例）＝営業利益＋受取利息＋受取配当金（＝金融収益）

### リターン分析
- 標準偏差（標本）の計算：√{ Σ(xi-x̄)^2 / (n-1) }
- 相関係数の計算：Σ{(xi-x̄)(yi-ȳ)} / √{ Σ(xi-x̄)^2 × Σ(yi-ȳ)^2 }  
    - 共分散を使った場合：YとZの共分散÷（Yの標準偏差×Zの標準偏差）

## 第3問：タックスプランニング
### 法人税の計算
- 交際費の損金
    - 交際費等（原則）は損金不算入（一定の範囲で損金算入できる例外あり）。
    - 飲食費のうち、**1人当たり10,000円以下**で一定の事項を記載した書類を保存するものは、交際費等から除外（会議費等として処理）。
    - それ以外の交際費等について、中小法人は「年800万円までの定額控除（全額損金算入）」または「接待飲食費の50%損金算入」等の範囲で損金算入（選択適用）。
- 源泉徴収税額は所得金額に加算せず、確定税額から差し引く（前払税額として税額控除）。
- 給与等の支給額が増加した場合の法人税額の特別控除（賃上げ促進税制）

| 区分 | 基本要件（賃上げ率） | 基本控除率 | 上乗せ（最大） | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| **中小企業者等** | **1.5%以上** | 15% | 2.5%増(+15%)、教育訓練(+10%)、くるみん/えるぼし等(+5%) → **最大45%** | 調整前法人税額の20%が上限、**控除しきれない分は5年繰越可** |
| **中堅企業（特定法人）** | **3%以上** | 10% | （全企業向けと同様）→ **最大35%** | 調整前法人税額の20%が上限 |
| **全企業** | **3%以上** | 10% | 賃上げ率（4%/5%/7%）や教育訓練等の加算 → **最大35%** | 調整前法人税額の20%が上限（一定の大企業は方針公表等が要件に追加） |

## 第4問：不動産
### 建蔽率と容積率の計算
1. 建蔽率
    - セットバックを考慮した新しい面積を計算する
    - 指定建蔽率80％である防火地域に、耐火建築物を建てるのであれば、建蔽率は100％になる
2. 容積率
    - 特定道路による緩和（前面道路幅員の加算）
        - 条件：特定道路(15m以上)が70m以内にあり、前面道路が6m〜12m未満の場合
        - 加算幅 Wa ＝ (12 － 前面道路幅員) × (70 － 距離L) / 70
        - 計算用幅員 ＝ 前面道路幅員 ＋ 加算幅 Wa
        - ※容積率は「指定容積率」と「(計算用幅員 × 係数)」の**いずれか小さい方**
            - 係数：**住居系は 0.4** ／ **商業・工業系は 0.6**

### 建築基準法
1. 斜線制限
    - **道路斜線制限**：**全ての用途地域**で適用される。
    - **隣地斜線制限**：**低層住居専用地域等以外**で適用される（高さ20mまたは31mを超える部分）。※低層住居専用地域には絶対高さ制限があるため適用外。
    - **北側斜線制限**：**低層住居専用地域**、**中高層住居専用地域**で適用される。
2. 高さ制限
    - **絶対高さ制限**：**10m または 12m**（低層住居専用地域・田園住居地域）
    - **隣地斜線制限**：対象は「20m」または「31m」を超える部分
        - **住居系**（低層等除く）：**20m**
        - **商業・工業系**：**31m**
    - **北側斜線制限**：低層住居専用地域、中高層住居専用地域に適用
3. 日陰規制
    - **低層住居専用地域**：軒高7m超 または 地上3階建以上
    - **その他（中高層、住居、準工業）**：高さ10m超
    - **商業、工業地域**：原則対象外
    - **例外**：規制外の10m超の建物が規制区域内に影を落とす場合は規制対象
### 譲渡所得の計算
1. 相続財産に係る譲渡所得の課税の特例（取得費加算の特例）
    - **内容**: 納付した相続税額のうち、譲渡した財産に対応する分を取得費に加算できる。
    - **期間要件**: 相続開始の翌日から相続税の申告期限の翌日以後**3年以内**（実質**3年10ヶ月以内**）の譲渡。
    - **併用**: 居住用3,000万円控除とは併用可。**「空き家特例」とは併用不可**。
2. 居住用財産を譲渡した場合の3,000万円の特別控除
    - **内容**: 譲渡益から最大3,000万円を控除。所有期間の長短は問わない。
    - **要件**: 配偶者や直系血族など特別関係者への譲渡は不可。前年・前々年に同種の特例等の適用がある場合は不可。
3. 居住用財産を譲渡した場合の長期譲渡所得の課税の特例（10年超軽減税率）
    - **条件**: 売却した年の1月1日時点で所有期間が**10年超**。
    - **内容**: 3,000万円控除と**重複適用が可能**。
    - **税率**: 6,000万円以下の部分＝**14.21%**（所得税10.21% + 住民税4%）、6,000万円超＝20.315%。
4. 取得費がわからない場合は、譲渡価額の**5％**（概算取得費）。
    - 実際の取得費が5%を下回る場合や、古い土地で不明な場合に使用。

## 第5問：相続・事業承継
### 類似業種比準価額の計算
**公式（概形）**: A × [(b/B + c/C + d/D) ÷ 3] × 斟酌率 ×（**1株当たり資本金等の額 ÷ 50円**）

1. **類似業種の数値（A, B, C, D）**を国税庁の資料から取得
2. **評価会社の数値（b, c, d）**を計算
    - b：1株当たり配当金額（過去2年平均）
    - c：1株当たり利益金額（過去2年平均）
    - d：1株当たり簿価純資産額
    - ※「1株当たり資本金等の額」が50円でない場合、50円換算の調整（倍率調整）を行う
3. **比準割合**：(b/B + c/C + d/D) ÷ 3
4. **斟酌率**：大会社0.7、中会社0.6、小会社0.5
5. **資本金調整**：最後に（**1株当たり資本金等の額 ÷ 50円**）を掛ける

### 会社規模別の株式評価方法

| 会社規模 | 評価方法 | 斟酌率 | 類似業種比準の係数 | 純資産価額の係数 |
| :--- | :--- | :---: | :---: | :---: |
| **大会社** | 類似業種比準のみ | **0.7** | 1.0 | 0 |
| **中会社の大** | 併用方式 | **0.6** | **0.90** | 0.10 |
| **中会社の中** | 併用方式 | **0.6** | **0.75** | 0.25 |
| **中会社の小** | 併用方式 | **0.6** | **0.60** | 0.40 |
| **小会社** | 純資産価額方式（原則） | — | 0 | 1.0 |

**計算の流れ：**
1. **斟酌率**で類似業種比準価額を計算
2. 計算した類似業種比準価額と純資産価額に、それぞれの**係数**を掛けて合計する

**例（中会社の大）：**
- 類似業種比準価額（斟酌率0.6で計算）× 0.90 ＋ 純資産価額 × 0.10

### 例外など（特定の評価会社）
**原則**: いずれも純資産価額方式で評価

1. **土地保有特定会社**
    - 判定：土地等の割合が、大会社70%以上、中会社90%以上
    - 評価：純資産価額のみ
2. **比準要素数1の会社**
    - 判定：配当・利益・純資産のうち2つが0（直前期＋直前々期）
    - 評価：純資産価額のみ（または選択で「類似業種0.25+純資産0.75」）
3. その他：株式等保有特定会社、開業後3年未満、開業前・休業中、清算中

### 相続時について（事業承継税制）
非上場株式等についての相続税の納税猶予及び免除の特例

| 項目 | 一般措置 | 特例措置 |
| :--- | :--- | :--- |
| 適用期限 | なし | 贈与・相続の適用期限：**2027/12/31**（特例承継計画の提出期限：**2026/3/31**） |
| 対象株式 | 2/3まで | **全株式（100%）** |
| 猶予割合 | 相続80%、贈与100% | **相続・贈与100%** |
| 後継者人数 | 1人 | **最大3人** |
| 雇用維持 | 5年平均8割 | 弾力化（理由報告で継続可） |

**免除されるケース**: 後継者死亡、次世代への贈与、会社破産など

**手続きの流れ:**
1. **特例承継計画の提出**（特例措置のみ、※2026年3月末まで延長）
2. **相続開始後8ヶ月以内**：都道府県から「円滑化法」の認定取得
3. **相続税申告期限（10ヶ月以内）まで**：税務署に納税猶予の申告書を提出、担保を提供
4. **継続届出**：申告期限から5年間は毎年、その後は3年ごとに提出

**適用後に守るべき要件:**

| 要件 | 一般措置 | 特例措置 |
| :--- | :--- | :--- |
| **雇用維持**（5年間） | 平均8割必須<br>未達成で打ち切り | 弾力化<br>未達成でも理由報告で継続可 |
| **代表者維持** | 後継者が代表でなくなった場合は打ち切り | 同左 |
| **議決権維持** | 後継者＋同族で50%超を維持 | 同左 |
| **筆頭株主維持** | 後継者が筆頭株主であり続ける | 同左 |

**主な取消事由（猶予が打ち切られるケース）:**
- 後継者が代表者でなくなった（やむを得ない理由を除く）
- 雇用維持要件未達成（一般措置のみ、特例措置は弾力化）
- 後継者＋同族の議決権が50%以下になった
- 納税猶予対象の株式を譲渡した
- 会社が解散・合併した
- 継続届出書を提出しなかった
- 会社が資産管理会社になった
- 本業の総収入が0円になった
