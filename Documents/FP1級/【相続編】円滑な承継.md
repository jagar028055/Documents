# 【FP1級取得への道 第7回】相続編：円滑な承継～相続税対策と事業承継の極意

## はじめに：相続は人生の総決算

FP1級取得に向けた連載第7回。今回は「相続編」として、円滑な承継のための相続税対策と事業承継について詳しく解説します。

なぜ相続対策が重要なのか？相続は単に財産を渡すだけでなく、人生の総決算であり、次世代への思いを託す重要なプロセスです。適切な準備なくして、円滑な承継は難しいでしょう。

特に2025年からの相続税・贈与税制度の大幅改正を控え、今こそ専門的な知識に基づいた相続計画が求められています。FPとして、お客様の人生を支える相続コンサルティング能力は、最高レベルの専門性が要求されます。

## 2025年相続税・贈与税制度改革のポイント

### 主要な改正内容

1. **相続時精算課税制度の見直し**
   ```
   現行：基礎控除110万円、累計2,500万円まで非課税
   改正：基礎控除が110万円減算（実質0円に）
   効果：暦年課税のメリットが相対的に上昇
   ```

2. **生前贈与加算期間の延長**
   ```
   現行：相続前3年間の贈与を加算
   改正：相続前7年間の贈与を加算
   影響：早期の贈与計画が重要に
   ```

3. **教育資金贈与制度の見直し**
   - 受贈者の年齢要件の変更
   - 使途の制限の明確化

4. **住宅取得資金贈与の縮小**
   - 非課税枠の段階的縮小
   - 消費税率の引下げ影響

### FPとしての対応

1. **既存プランの見直し**
   - 相続時精算課税選択者の再検討
   - 贈与計画の期間延長対応

2. **新制度への移行支援**
   - 暦年課税の活用提案
   - 早期贈与の重要性説明

## 相続税の基本：正確な理解

### 相続税の計算プロセス

**Step1：課税価格の計算**
```
各人の課税価格 = 取得財産 + みなし相続財産
               - 債務・葬億費用
               + 相続開始前3年以内の贈与財産
```

**Step2：課税遺産総額の計算**
```
課税遺産総額 = 各人の課税価格の合計 - 基礎控除額
基礎控除額 = 3,000万円 + 600万円 × 法定相続人数
```

**Step3：相続税の総額計算**
```
法定相続分に応ずる取得金額を算出
↓
各人の算出税額を計算（速算表使用）
↓
各人の算出税額を合算して相続税総額
```

**Step4：各人の納税額計算**
```
各人の納税額 = 相続税総額 × 実際の取得割合
            - 各種税額控除
```

| 法定相続分に応ずる取得金額 | 税率 | 控除額 |
|---|---|---|
| ～1,000万円 | 10% | 0円 |
| ～3,000万円 | 15% | 50万円 |
| ～5,000万円 | 20% | 200万円 |

## 事業承継：後継者育成と資金確保

### 事業承継の現状と課題

**深刻化する後継者不足**
- 70歳以上の経営者：約243万人
- 後継者が決まっていない：約60%
- 2025年問題：団塊の世代の大量引退

**事業承継の3つの方法**
1. **親族内承継**
   - メリット：事業の継続性、従業員の雇用維持
   - デメリット：適格な後継者の必要性、多額の相続税

2. **第三者への承継（M&A）**
   - メリット：高額な売却益、事業存続の可能性
   - デメリット：従業員の不安、ブランドの変化

3. **従業員への承継（ESOP）**
   - メリット：従業員のモチベーション向上
   - デメリット：資金力の問題、経営能力

### 事業承継税制の活用

**特例の概要**
```
適用要件
├── 先代経営者からの贈与
├── 贈与税の猶予
├── 後継者の経営参画（5年超）
└── 雇用維持（8割超）

猶予税額の全額免除
├── 贈与者死亡時
├── 経営下の期間が5年超
└── 雇用維持を継続
```

**適用ステップ**
1. 事業承継計画の策定
2. 経営構造改善計画の提出
3. 都道府県知事の確認
4. 税務署長への承認申請

## 遺言の種類と活用

### 遺言の5つの種類

1. **自筆証書遺言**
   - 全文を自筆で作成
   - 日付・署名・押印が必要
   - 2020年から法務局保管制度開始

2. **公正証書遺言**
   - 公証人が作成
   - 公正役場での作成
   - 原本は公証役場で保管

3. **秘密証書遺言**
   - 内容を秘密にして作成
   - 公証人への封印と確認

4. **緊急時遺言**
   - 病気・怪我などの場合
   - 証人3人の立会い必要

5. **船舶遺言**
   - 船舶中での作成

### 遺言書作成のポイント

**記載すべき重要事項**
1. **財産の指定**
   - 誰に何を渡すか
   - 具体的な特定方法

2. **遺産分割の方法**
   - 分割の指定
   - 分割禁止の定め

3. **遺言執行者の指定**
   - 信頼できる人物
   - 専門家の任命

4. **相続人の廃除**
   - 相続権の剥奪
   - 家庭裁判所の確認

## 信託の活用：柔軟な財産管理

### 家族信託の基本

**信託の仕組み**
```
委託者（財産の所有者）
↓財産の信託
受託者（財産の管理者）
↓受益の給付
受益者（利益を受ける人）
```

**家族信託のメリット**
- 認知症などに備えることができる
- 柔軟な財産管理が可能
- 相続手続きの簡素化
- 遺言の内容を確実に実行

### 家族信託の活用例

1. **認知症対策**
   ```
   委託者：高齢の親
   受託者：成人した子供
   受益者：委託者（生存中）、子供（死後）
   ```

2. **障がいのある子供の支援**
   ```
   親死亡後も財産管理を継続
   生活費の定期的な給付
   公的給付との調整
   ```

3. **相続税対策**
   ```
   二次相続の税負担軽減
   財産の分割所有
   節税効果の最大化
   ```

### 不動産信託の活用

**不動産信託の特徴**
- 登記手続きが必要
- 信託登記費用がかかる
- 管理の専門性が必要

**活用場面**
- 複数の相続人間での調整
- 不動産の有効活用
- 譲渡所得税の対策

## 節税対策：実践的な手法

### 生前贈与の活用

1. **暦年贈与（基礎控除110万円）**
   ```
   年間110万円までは贈与税がかからない
   長期間のコツコツ贈与が有効
   2025年からは7年前の贈与が加算対象
   ```

2. **相続時精算課税**
   ```
   65歳以上の親から20歳以上の子へ
   累計2,500万円まで非課税
   超過分は20%の贈与税
   相続時に贈与財産を加算
   ```

3. **配偶者控除（2,110万円）**
   ```
   20年以上婚姻期間
   居住用不動産の贈与
   基礎控除110万円と合計2,110万円非課税
   ```

### 生命保険の活用

**相続税対策としての生命保険**
1. **納税資金の確保**
   - 死亡保険金で相続税を支払う
   - 資産の売却を避ける

2. **相続税の非課税枠**
   ```
   法定相続人1人あたり500万円非課税
   例：相続人3人の場合、1,500万円まで非課税
   ```

3. **代償分割の資金**
   - 特定の相続人が不動産を取得
   - 他の相続人に保険金を分配

### 小規模宅地等の特例

**適用要件と減額割合**
| 区分 | 面積要件 | 減額割合 |
|---|---|---|
| 特定居住用宅地等 | 330㎡以下 | 80% |
| 特定事業用宅地等 | 400㎡以下 | 80% |
| 貸付用宅地等 | 200㎡以下 | 50% |

**活用ポイント**
- 被相続人の居住用宅地
- 生計同一の親族が取得
- 相続税申告期限まで保有
