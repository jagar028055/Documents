# 【FP1級取得への道 第5回】税務編：賢い節税戦略～税金の知識で資産を守る

## はじめに：税金は資産形成の要

FP1級取得に向けた連載第5回。今回は「税務編」として、税金に関する知識と節税戦略について詳しく解説します。

なぜ税務知識が重要なのか？資産運用で得た利益も、税金の知識がなければ目減りしてしまいます。適切な税務対策は、資産形成の効果を最大化し、合法的に税負担を軽減するための必須スキルです。

FPとしてお客様の資産管理を支援するためには、複雑な税制度を理解し、最適な節税提案ができる高度な専門知識が求められます。

## 所得税：正確な理解と節税の基本

### 所得税の基本構造

**所得税の計算フロー**
```
1. 収入金額の合計
↓
2. 必要経費の控除（給与所得控除等）
↓
3. 所得金額の算出（10種類の所得）
↓
4. 所得控除（14種類）
↓
5. 課税所得金額
↓
6. 税額の計算（累進課税）
↓
7. 税額控除
↓
8. 納付すべき所得税額
```

**所得税の種類と特徴**

| 所得の種類 | 計算方法 | 主な源泉 |
|---|---|---|
| 利子所得 | 収入金額 = 所得金額 | 預貯金の利子 |
| 配当所得 | 収入金額 - 負債利子 | 株式の配当 |
| 不動産所得 | 総収入金額 - 必要経費 | 家賃収入 |
| 事業所得 | 総収入金額 - 必要経費 | 個人事業の利益 |
| 給与所得 | 収入金額 - 給与所得控除 | 給料・賞与 |
| 譲渡所得 | 総収入金額 - 必要経費 | 資産の売却益 |
| 一時所得 | 総収入金額 - 必要経費 - 特別控除 | 賞金・懸賞 |
| 雑所得 | 上記以外の所得 | 公的年金等 |
| 山林所得 | 総収入金額 - 必要経費 - 特別控除 | 山林の譲渡 |
| 退職所得 | （収入金額 - 退職所得控除） ÷ 2 | 退職金 |

### 所得控除の種類と活用法

**人的控除（所得金額から控除）**

| 控除の種類 | 控除額 | 要件 |
|---|---|---|
| 基礎控除 | 48万円 | 全員 |
| 配偶者控除 | 13万～48万円 | 配偶者の所得48万円以下 |
| 扶養控除 | 13万～63万円 | 扶養親族の年齢により変動 |
| 寡婦控除 | 27万円 | 夫と死別・離婚、所得500万円以下 |
| 障害者控除 | 27万～75万円 | 障害者の等級により変動 |

**物的控除（所得金額から控除）**

| 控除の種類 | 控除額 | 要件 |
|---|---|---|
| 社会保険料控除 | 支払った全額 | 健康保険、厚生年金等 |
| 生命保険料控除 | 最高12万円 | 新契約・旧契約で計算 |
| 地震保険料控除 | 最高5万円 | 地震保険料の支払い |
| 医療費控除 | 最高200万円 | 10万円超の医療費支払い |
| 寄付金控除 | 支払額-2,000円 | ふるさと納税等 |

### FP1級で求められる所得税対策

1. **給与所得者の節税ポイント**
   ```
   扶養控除の最大活用
   ├── 16～18歳：特定扶養親族 63万円
   ├── 19～22歳：特定扶養親族 63万円
   └── 23～69歳：一般扶養親族 38万円

   配偶者控除の最適化
   ├── 配偶者の所得48万円以下：最大48万円
   └── 配偶者特別控除：最大38万円（95万円以下）
   ```

2. **医療費控除の活用法**
   ```
   対象となる医療費
   ├── 治療費、薬代
   ├── 通院交通費（公共機関）
   ├── 医療用器具
   └── 介護保険・介護サービス

   対象外となるもの
   ├── 美容整形
   ├── 健康診断（異常なし）
   ├── ビタミン剤
   └── 自家用車での通院
   ```

3. **ふるさと納税の最適化**
   ```
   自己負担2,000円で控除される限度額
   =（総所得金額 - 人的控除）× 10% - 2,000円

   節税効果の例
   年収600万円：上限約8万円
   年収800万円：上限約12万円
   年収1,000万円：上限約17万円
   ```

## 相続税：賢い相続対策の実践

### 相続税の基本

**相続税の計算フロー**
```
1. 正味の遺産額（相続財産 - 債務・葬億費用）
↓
2. 課税遺産総額（正味遺産額 - 基礎控除額）
↓
3. 相続税の総額（法定相続分で按分して計算）
↓
4. 各人の納税額（実際の取得割合で配分）
```

**基礎控除額**
```
基礎控除額 = 3,000万円 + 600万円 × 法定相続人数
例：相続人3人の場合
3,000万円 + 600万円 × 3 = 4,800万円
```

**相続税の税率（速算表）**

| 法定相続分に応ずる取得金額 | 税率 | 控除額 |
|---|---|---|
| ～1,000万円 | 10% | 0円 |
| ～3,000万円 | 15% | 50万円 |
| ～5,000万円 | 20% | 200万円 |
| ～1億円 | 30% | 700万円 |
| ～2億円 | 40% | 1,700万円 |
| ～3億円 | 45% | 2,700万円 |
| ～6億円 | 50% | 4,200万円 |
| 6億円超 | 55% | 7,200万円 |

### 相続税の評価方法

**土地の評価**
1. **路線価方式**
   - 面向する道路の路線価 × 面積
   - 奥行価格補正、角地補正等

2. **倍率方式**
   - 固定資産税評価額 × 倍率

**建物の評価**
- 固定資産税評価額 × 1.0

**株式の評価**
1. **上場株式**
   - 課税時期の終値
   - 月平均額、終値月平均額

2. **非上場株式**
   - 類似業種比準価法
   - 純資産価額法
   - 配当還元法

### 相続税対策の具体策

**生前対策（暦年課税を活用）**

1. **暦年贈与**
   ```
   年間110万円の基礎控除を活用
   孫への贈与：相加対象外
   教育資金贈与：最大1,500万円非課税
   結婚・子育て資金：最大1,000万円非課税
   ```

2. **配偶者への贈与**
   ```
   居住用不動産の贈与
   ├── 基礎控除110万円
   ├── 配偶者控除2,000万円
   └── 合計2,110万円まで非課税

   注意点：相続時精算課税との選択
   ```

**不動産を活用した対策**

1. **小規模宅地等の特例**
   ```
   居住用宅地
   ├── 特定居住用宅地等：330㎡まで80%減額
   └── 特定事業用宅地等：400㎡まで80%減額

   貸付用宅地
   └── 特定同族会社事業用宅地等：400㎡まで80%減額
   ```

2. **賃貸アパート経営**
   ```
   メリット
   ├── 不動産所得による相続税納税資金の確保
   ├── 小規模宅地等の特例適用
   └── 収益物件によるキャッシュフロー

   デメリット
   ├── 入居率変動リスク
   ├── 管理の手間
   └── 固定資産税の増加
   ```

## 贈与税：上手な活用術

### 贈与税の基本

**贈与税の計算**
```
贈与税額 = （年間贈与額 - 基礎控除110万円） × 税率
```

**贈与税の税率（速算表）**

| 基礎控除後の贈与額 | 65歳未満 | 65歳以上 |
|---|---|---|
| ～200万円 | 10% | 10% |
| ～300万円 | 15% | 15% |
| ～400万円 | 20% | 20% |
| ～600万円 | 30% | 20% |
| ～1,000万円 | 40% | 30% |
| ～1,500万円 | 45% | 40% |
| ～3,000万円 | 50% | 45% |
| 3,000万円超 | 55% | 50% |

### 相続時精算課税制度

**制度概要**
```
65歳以上の親から20歳以上の子への贈与
・累計2,500万円まで贈与税が非課税
・超過額は20%の贈与税
・相続時に贈与財産を相続財産に加算
```

**活用ポイント**
- 大きな不動産の移転に有利
- 将来値上がりが見込む資産に有利
- 相続税がかからない可能性がある場合

## 不動産所得の税務

### 不動産所得の計算

```
不動産所得 = 総収入金額 - 必要経費
```

**総収入金額に含まれるもの**
- 家賃、礼金、権利金
- 更新料、敷金（返還不要分）
- 共益費、管理費

**必要経費に含まれるもの**
- 固定資産税、都市計画税
- 減価償却費
- 修繕費、管理費
- 火災保険料、損害保険料
- 借入金の利子

### 青色申告の特典

**要件**
- 不動産所得の年間収入が300万円を超える
- 帳簿作成ができる
- 税務署に届出書を提出

**特典**
- 青色申告特別控除：最大65万円
- 青色事業専従者給与：配偶者最大86万円、その他50万円
- 純損失の繰越控除（3年間）

## 譲渡所得の税務

### 不動産譲渡所得

**計算式**
```
譲渡所得 = 譲渡価格 - （取得費 + 譲渡費用）
```

**取得費の計算**
```
取得費 = 購入代価 + 購入諸費用 - 減価償却費
```

**軽減税率の特例**
1. **居住用財産の3,000万円特別控除**
   - 居住期間10年以上
   - 譲渡価格から3,000万円控除

2. **10年以上所有の軽減税率**
   - 課税長期譲渡所得金額
   - 6,000万円以下：10%
   - 6,000万円超：15%

### 株式譲渡所得

**税率**
- 上場株式：20.315%（所得税15%+復興特別所得税0.315%+住民税5%）
- 未上場株式：20.42%（所得税15%+復興特別所得税0.42%+住民税5%）

**特定口座の活用**
- 源泉徴収選択口座：確定申告不要
- 簡易申告口座：確定申告が必要

## FP1級試験での出題傾向

### 過去問分析

**出題パターン**：
1. 所得税計算問題（约30%）
2. 相続税計算問題（约30%）
3. 贈与税・相続時精算課税問題（约20%）
4. 節税対策の事例問題（约20%）

**頻出計算問題**：
- 給与所得の所得税計算
- 医療費控除の計算
- ふるさと納税の限度額計算
- 相続税の総額計算
- 不動産譲渡所得の計算

**注意すべきポイント**：
- 千円未満の切り捨て
- 控除額の上限
- 特例の適用要件

### 効率的な学習方法

1. **計算問題の反復練習**
   - 典型的な計算パターンの暗記
   - よくある数字の記憶
   - 計算スピードの向上

2. **制度の体系的理解**
   - 各税法の比較表作成
   - 控除・特例の一覧表化
   - 適用要件の整理

3. **事例問題の対策**
   - よくある節税シナリオの学習
   - 複数の制度を組み合わせた対策
   - 実務での活用方法

## 実務での税務相談

### 相談の進め方

1. **ヒアリング**
   - 家族構成の確認
   - 資産・負債状況
   - 年間収支の把握
   - 将来の計画

2. **分析**
   - 課税関係の特定
   - 節税の可能性検討
   - リスクの確認

3. **提案**
   - 具体的な節税策の提示
   - 実行ステップの明示
   - 注意点の説明

### 成功事例のポイント

1. **具体的で実行可能な提案**
   - 短期的に実現できる対策
   - 費用対効果の考慮
   - 家族の理解を得る提案

2. **継続的なフォロー**
   - 制度改正への対応
   - 状況変化に応じた見直し
   - 成果の確認

## まとめ：税務知識はFPの専門性の証

税務知識はFPの専門性を示す重要な要素です。複雑な税制度を理解し、お客様の最適な節税を提案することで、真の価値を発揮できます。

**重要なポイントの再確認**：
1. 所得税の控除・特例を最大限活用する
2. 相続税は早期から計画的に対策する
3. 贈与税は制度の特性を理解して活用する
4. 不動産・譲渡所得は適切な経費計上と特例活用
