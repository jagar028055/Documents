# 【FP1級取得への道 最終回】実践編：ケーススタディ～総合力で挑む実務レベルの課題

## はじめに：知識の総動員と実践力の養成

FP1級取得に向けた連載の最終回です。これまで基礎から専門分野まで幅広く学んできましたが、最後は総合的な実践力を養うことが重要です。

本記事では、実際のFP業務で遭遇する典型的なケーススタディを通じて、これまで学んだ知識を総動員し、問題解決能力を高めていきましょう。

FP1級の実技試験では、単なる知識の暗記だけでなく、お客様の状況を正確に分析し、最適なプランを提案する実践的な能力が求められます。本記事を通じて、その実践力を身につけましょう。

## FP業務の基本フロー

### 相談から実行までの5ステップ

1. **ヒアリングと情報収集**
   - 家族構成の確認
   - 資産・負債の把握
   - 収支状況の分析
   - 将来の希望と不安の確認

2. **現状分析と問題点の特定**
   - 財務諸表の作成
   - 各種指標の計算
   - リスクの洗い出し
   - 課題の優先順位付け

3. **目標設定とプランニング**
   - 具体的な目標設定
   - 複数のプラン作成
   - 各プランの評価
   - 最適プランの選択

4. **提案と合意形成**
   - 分かりやすい説明
   - 質疑応答への対応
   - 家族の合意形成
   - 実行計画の策定

5. **実行とフォロー**
   - プランの実行支援
   - 定期的な進捗確認
   - 必要に応じた修正
   - 長期的な関係構築

## ケーススタディ1：若年家族のライフプランニング

### 基本情報

```
ご依頼者：山田さん（32歳・男性）
家族構成：妻（30歳）・長女（3歳）・長男（0歳）
職業：IT企業勤務（年収800万円）
住居：分譲マンション（住宅ローン残2,500万円）
資産：預貯金200万円・投資信託100万円
```

### 相談内容と希望

「将来の子供たちの教育資金と、我々夫婦の老後資金について、どのように準備していけばよいか相談したい。特に、教育資金はいくら必要で、いつから始めればよいか知りたい。」

### STEP1：現状分析

**家計収支分析**
```
年収：800万円
├── 手取り収入：約620万円
├── 年間支出：540万円
│   ├── 住宅ローン返済：180万円
│   ├── 生活費：300万円
│   └── 保険料：60万円
└── 年間貯蓄：80万円（貯蓄率12.9%）
```

**財務状況分析**
```
資産：300万円
├── 預貯金：200万円
└── 投資信託：100万円
負債：2,500万円
└── 住宅ローン：2,500万円
純資産：△2,200万円
```

### STEP2：課題の特定

1. **教育資金の準備不足**
   - 大学までの教育費：約2,000万円必要
   - 現状のままでは準備が間に合わない

2. **貯蓄率が低い**
   - 目標貯蓄率20%に対し、現状12.9%
   - 毎月50万円の貯蓄増が必要

3. **資産運用が不十分**
   - 預貯金比率が高い（67%）
   - 運用リターンの機会損失

4. **保険の見直しが必要**
   - 保険料が月5万円と高額
   - 必要保障の再検討

### STEP3：プランニング

**教育資金準備プラン**

```
子供1人あたりの教育費
├── 幼稚園～高校：約500万円
├── 大学（私立文系）：約500万円
└── 合計：約1,000万円

準備方法
├── 学資保険：月3万円×18年＝650万円
└── 積立投資：月2万円×15年＝360万円
```

**貯蓄増額プラン**
```
支出削減
├── 通信費：年間6万円削減
├── 外食費：年間12万円削減
├── 保険見直し：年間24万円削減
└── 合計：年間42万円削減

収入増加
├── 賞与：年間30万円増加見込み
└── 合計：年間72万円貯蓄増加可能
```

**資産運用プラン**
```
iDeCo：月2.3万円（年27.6万円）
├── 節税効果：約6万円
├── 20年後：約1,100万円（年利3%で）
└── 老後資金の一部として活用

つみたてNISA：月3万円（年36万円）
├── 非課税効果：最大
├── 18年後：約1,100万円（年利5%で）
└── 教育資金の一部として活用
```

### STEP4：提案のポイント

1. **優先順位の明確化**
   - まず教育資金の確保
   - 次に老後資金の準備
   - 住宅ローンは現状維持

2. **実行可能性の考慮**
   - 無理のない範囲での改善
   - 長期的な視点での継続
   - 家族全体の理解

3. **定期的な見直し**
   - 年1回の進捗確認
   - 子供の成長に応じた調整
   - 収入変動への対応

## ケーススタディ2：高齢者の資産管理と相続対策

### 基本情報

```
ご依頼者：佐藤さん（75歳・男性）
家族構成：妻（73歳）・長男（45歳）・長女（42歳）
職業：元会社経営者（現在無職）
住居：一戸建て（評価額5,000万円）
資産：預貯金8,000万円・投資信託2,000万円
収入：年金年間300万円・不動産収入200万円
```

### 相談内容と希望

「自分たちが亡くなった後の相続について、スムーズに進むように準備したい。特に、相続税をできるだけ抑えたい。また、妻の今後の生活も心配だ。」

### STEP1：現状分析

**収支分析**
```
年間収入：500万円
├── 年金：300万円
└── 不動産収入：200万円

年間支出：400万円
├── 生活費：250万円
├── 医療費：50万円
├── 税金：50万円
└── その他：50万円

年間収支：100万円黒字
```

**資産評価**
```
資産：1億5,000万円
├── 預貯金：8,000万円
├── 投資信託：2,000万円
├── 自宅：5,000万円
└── その他：0円

負債：0円
純資産：1億5,000万円
```

### STEP2：相続試算

**相続税計算**
```
課税遺産総額：1億5,000万円 - 4,200万円（基礎控除）
            = 1億800万円

相続税総額：約2,800万円

法定相続分での取得
├── 妻：2分の1（1,400万円）
├── 長男：4分の1（700万円）
└── 長女：4分の1（700万円）
```

**妻の生活資金試算**
```
必要老後資金
├── 平均寿命まで：約15年
├── 年間生活費：300万円
└── 合計：4,500万円

現状の資産
├── 法定相続分：7,500万円
├── 生命保険：1,000万円
└── 合計：8,500万円（十分な資金あり）
```

### STEP3：対策の検討

**相続税対策**
```
1. 生前贈与の活用
   ├── 孫への贈与：年110万円×2人＝220万円
   ├── 教育資金贈与：1,000万円
   └── 結婚・子育て資金：1,000万円

2. 配偶者控除の活用
   ├── 居住用不動産の贈与
   ├── 最大2,110万円まで非課税
   └── 時期を検討

3. 生命保険の活用
   ├── 納税資金の確保
   ├── 非課税枠：500万円×3人＝1,500万円
   └── 新規加入を検討
```

**妻の生活確保**
```
1. 遺言による配分指定
   ├── 妻に多めの遺産を配分
   ├── 遺産分割協議の回避
   └── 遺留分に配慮

2. 家族信託の活用
   ├── 資産管理の委任
   ├── 生活費の定期的な給付
   └── 認知症への備え
```

### STEP4：提案のポイント

1. **段階的な実行**
   - まず生前贈与の開始
   - 次に生命保険の検討
   - 最後に遺言書の作成

2. **家族との合意形成**
   - 子供たちとの事前相談
   - 公平感のある配分
   - 感情的対立の回避

3. **専門家の活用**
   - 税理士による税務相談
   - 弁護士による遺言作成
   - 司法書士による登記手続き

## ケーススタディ3：事業承継と資金確保

### 基本情報

```
ご依頼者：鈴木さん（68歳・男性）
事業：製造業（年商5億円）
家族構成：妻（65歳）・長男（40歳）
資産：会社資産3億円・個人資産1億円
後継者：長男（役員在籍20年）
```

### 相談内容と希望

「会社を長男に承継させたいが、相続税が多額になるため、効果的な対策を知りたい。また、承継後の会社運営も安定させたい。」

### STEP1：現状分析

**会社評価**
```
株式評価額：3億円
├── 純資産価値：2億円
└── 類似業種比準価：4億円
└── 評価額：3億円（中間値）
```

**個人資産**
```
資産：1億円
├── 預貯金：5,000万円
├── 自宅：4,000万円
└── その他：1,000万円
```

**相続試算**
```
課税遺産総額：4億円 - 3,600万円（基礎控除）
            = 3億6,400万円

相続税総額：約1億2,000万円

長男の負担
├── 株式取得：3億円
└── 相続税：6,000万円
合計：3億6,000万円
```

### STEP2：事業承継税制の検討

**適用要件の確認**
```
□ 先代経営者からの贈与
□ 後継者の経営参画5年以上
□ 雇用維持8割以上
□ 経営構造改善計画の提出
```

**適用効果**
```
贈与税の全額猶予
├── 贈与税：約6,000万円
└── 猶予期間：無期限

納税猶予の全額免除
├── 贈与者死亡時
├── 5年間の経営継続
└── 雇用維持の継続
```

### STEP3：資金確保プラン

**納税資金の確保**
```
1. 生命保険の活用
   ├── 法人役員保険：月50万円
   ├── 死亡保険金：1億円
   └── 社内貸付による返済

2. 事業用資産の売却
   ├── 不要土地の売却：5,000万円
   └── 資金繰りの改善

3. 銀行融資の活用
   ├── 事業承継融資
   └── 担保：株式・不動産
```

**安定経営のための対策**
```
1. 経営計画の策定
   ├── 中期経営計画
   ├── 資金計画
   └── 人材計画

2. 従業員の定着
   ├── ストックオプション
   ├── 退職金制度
   └── 教育訓練

3. 取引先との関係強化
   ├── 重要取引先の維持
   ├── 新規取引の開拓
   └── 品質管理の徹底
```

### STEP4：実行計画

**タイムスケジュール**
```
1年目
├── 経営構造改善計画策定
├── 事業承継税制の申請
└── 後継者育成の開始

2年目
├── 株式の一部譲渡
├── 生命保険の加入
└── 従業員への説明

3年目
├── 残り株式の譲渡
├── 経営権の移譲
└── 納税資金の確保
```

## FP1級実技試験対策

### 試験形式の理解

**試験時間**：2時間30分
**問題構成**：
- 事例問題（2問）
- 財務分析、保険、運用、税務、相続の各分野から出題

**評価ポイント**：
1. **分析力**（30%）
   - 状況把握の正確性
   - 数値分析の適切性

2. **提案力**（40%）
   - 具体的な提案内容
   - 実行可能性の考慮

3. **表現力**（20%）
   - 分かりやすい説明
   - 適切な専門用語

4. **倫理観**（10%）
   - FP倫理の遵守
   - 顧客利益最優先

### 時間配分のコツ

**解答時間配分**
```
問題1：60分
├── 読解と分析：15分
├── 解答構成：10分
├── 記述：30分
└── 見直し：5分

問題2：60分
├── 読解と分析：15分
├── 解答構成：10分
├── 記述：30分
└── 見直し：5分

見直し：30分
```

### 解答案の作成テクニック

1. **構成の明確化**
   ```
   Ⅰ. 現状認識
    1. 家族構成
    2. 資産・負債状況
    3. 収支状況

   Ⅱ. 課題の特定
    1. 財務上の問題点
    2. リスクの洗い出し
    3. 優先順位付け

   Ⅲ. 提案内容
    1. 短期的対策
    2. 中期的対策
    3. 長期的対策

   Ⅳ. 実行計画
    1. 具体的なステップ
    2. 時期と期間
    3. 注意点
   ```

2. **数字根拠の明示**
   - 計算過程の提示
   - 前提条件の明記
   - 比較検討の提示

3. **視覚的な工夫**
   - 箇条書きの活用
   - 表や図の作成
   - 重要箇所の強調

## 実務で活かすためのヒント

### コンサルティングスキルの向上

1. **傾聴力の養成**
   - 相手の話を遮らない
   - 背景にある不安の理解
   - 共感の姿勢を大切に

2. **質問力の向上**
   - オープンな質問
   - 具体的な質問

3. **説明力の磨き**
   - 専門用語を避ける

よくある失敗例と対策

1. **過度な期待を持たせる**
   - 対策：現実的な目標設定

2. **家族の合意がない**
   - 対策：家族全員との面談

3. **継続的なフォロー不足**
   - 対策：定期面談の約束

## まとめ：FPとしての成長へ

3. **倫理観の保持**：常に顧客利益を最優先に
4. **人間的な成長**：豊かな人間性と共感力
