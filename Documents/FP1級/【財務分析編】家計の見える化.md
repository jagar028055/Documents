# 【FP1級取得への道 第2回】財務分析編：家計の見える化～お客様の本当の姿を把握する技術

## はじめに：財務分析はFPの基本スキル

前回に続き、FP1級取得に向けた連載第2回です。今回は「財務分析編」として、家計の見える化について詳しく解説します。

なぜ財務分析が重要なのか？それは、お客様の現在地を正確に把握することなく、適切なライフプランを描くことができないからです。医者が診断なくして治療できないように、FPも財務分析なくして的確なアドバイスはできません。

本記事では、FP1級レベルで求められる高度な財務分析技術を、実務で活かせる形で徹底解説します。

## 財務諸表の基本：3つの重要な書類

FPが作成・分析する財務諸表は主に3つです。それぞれの役割と関連性を理解しましょう。

### 1. 家計収支表（キャッシュフロー表）

**目的**：一定期間の収入と支出の流れを把握

**重要な視点**：
- 黒字（貯蓄可能額）か赤字か
- 支出構造の特徴
- 変動費と固定費のバランス
- 貯蓄・投資への回転率

**FP1級で求められる分析レベル**：
- 単なる集計から「なぜそうなるのか」まで分析
- 将来の変動予測
- 改善可能性の具体化

### 2. 貸借対照表（バランスシート）

**目的**：特定時点の資産・負債の状況を把握

**重要な視点**：
- 純資産（正味資産）の額
- 資産の構成と流動性
- 負債の種類と返済計画
- 資産と負債のバランス

**FP1級で求められる分析レベル**：
- 資産配分の適切性評価
- 負債返済能力の分析
- 資産流動性の確保度評価

### 3. ライフプラン表

**目的**：人生イベント別の資金必要量と準備計画

**重要な視点**：
- ライフイベントごとの資金需要
- 資金不足・過剰の識別
- 公的保障の活用
- プランの現実性評価

## 家計収支表の作成：実践的ノウハウ

### STEP1：収入の正確な把握

**把握すべき収入の種類**

1. **安定収入**
   - 給与所得（手取り額）
   - 年金受給額
   - 不動産賃貸収入
   - 定期的な配当金・利子

2. **変動収入**
   - ボーナス・賞与
   - 臨時収入（相続、贈与）
   - 副業・フリーランス収入
   - 預貯金の利息・配当

**FP1級レベルの注意点**：
- 税引き後の手取り額で計算
- ボーナスは平均化ではなく実績ベースで
- 社会保険料控除後の額を正確に
- 変動収入は過去3年間の平均を参考

### STEP2：支出の詳細な分類

**支出カテゴリーの体系化**

```
固定費（変動が少ない支出）
├── 住居費
│   ├── 家賃・地代
│   ├── 住宅ローン返済（元金・利子）
│   └── 管理費・修繕積立金
├── 保険料
│   ├── 生命保険料
│   ├── 損害保険料
│   └── 社会保険料
├── 税・社会費
│   ├── 所得税・住民税
│   └── 固定資産税
└── その他固定費
    ├── 駐車場代
    ├── 共益費
    └── 支払い利息

変動費（状況に応じて変動する支出）
├── 食費
├── 水道光熱費
├── 通信費
├── 交通費
├── 日用品・衣服費
├── 教育費
├── 医療・保険外医療費
├── 交際費・娯楽費
├── その他雑費
└── 特別費
    ├── 旅行費
    ├── 趣味・娯楽
    └── 臨時の出費
```

**FP1級で求められる分類技術**：
- お客様の生活スタイルに合わせたカスタマイズ
- 将来の増減予測を組み込んだ分類
- 改善可能性の高い項目の識別

### STEP3：収支分析と改善点の発見

**分析の視点**

1. **収支構造の評価**
   - 貯蓄率（可処分所得に占める貯蓄の割合）
   - 固定費比率（固定費の収入に占める割合）
   - 変動費の推移
   - キャッシュフローの安定性

2. **ベンチマークとの比較**
   - 世帯収入階層別平均支出
   - 同じライフステージの平均
   - 理想的な貯蓄率（15%～20%が目標）

3. **改善可能性の具体化**
   - 短期的に改善可能な項目
   - 中期的な調整が必要な項目
   - 長期的な構造改革が必要な項目

## 貸借対照表の作成：資産負債の網羅的把握

### 資産の分類と評価

**流動資産（1年以内に現金化可能）**

```
現金・預貯金
├── 普通預金・当座預金
├── 定期預金
├── 貯蓄預金
├── 財形貯蓄・社内預金
└── 外貨預金

有価証券
├── 株式・投資信託
├── 債券
├── 国債・地方債
└── ETF・REIT

その他流動資産
├── 個人年金保険
├── 積立型終身保険
└── 短期的な貸付金
```

**固定資産（1年以上保有の資産）**

```
不動産
├── 土地
├── 建物（自宅・投資用）
├── マンション
└── 別荘・駐車場

投資不動産
├── 賃貸用不動産
├── 区分所有権
└── 投資用土地

その他固定資産
├── 貴金属・骨董品
├── 自動車・バイク
├── 耐久消費財
└── 事業用資産
```

**FP1級レベルの評価技術**：
- 時価評価の徹底（簿価ではなく）
- 流動性の段階的評価
- 資産間の関連性の把握

### 負債の正確な把握

**短期負債（1年以内返済）**

```
クレジットカード債務
├── リボ残高
├── 分割払い残高
└── カードローン

短期借入金
├── 友人・知人からの借入
├── サラ金・消費者金融
└── その他短期借入

その他短期負債
├── 未払い金
├── 前払い費用
└── 繰延べ税金資産
```

**長期負債（1年超返済）**

```
住宅ローン
├── フラット35
├── 民間金融機関
├── 財形融資
└── 親子リレー返済

教育ローン
├── 国の教育ローン
├── 民間教育ローン
└── 奨学金（本人返済分）

その他長期負債
├── 事業借入金
├── 自動車ローン
├── 不動産投資ローン
└── 法人借入（保証人付）
```

**FP1級で求められる分析技術**：
- 返済負担率の正確な計算
- 金利リスクの評価
- 早期返済効果のシミュレーション

## 分析指標：数字から読み解く健康度

### 1. 収支関連指標

**貯蓄率**
```
貯蓄率 = （年間貯蓄額 ÷ 可処分所得）× 100
理想値：15%以上、目標20%
```

**可処分所得率**
```
可処分所得率 = （手取り収入 ÷ 総収入）× 100
一般的に70～80%
```

**固定費比率**
```
固定費比率 = （固定費 ÷ 可処分所得）× 100
理想値：50%以下
```

### 2. 資産負債関連指標

**純資産率**
```
純資産率 = （純資産 ÷ 総資産）× 100
理想値：60%以上
```

**流動性比率**
```
流動性比率 = （流動資産 ÷ 流動負債）× 100
理想値：200%以上
```

**返済負担率**
```
年間返済負担率 = （年間ローン返済額 ÷ 年間手取り収入）× 100
理想値：25%以下（35%以上は要注意）
```

**負債資産倍率**
```
負債資産倍率 = （総負債 ÷ 総資産）
理想値：0.4以下
```

### 3. ライフプラン関連指標

**ライフプーン達成可能性**
- 必要資産額の充足率
- 現在の貯蓄率での達成年数
- リタイアメント準備度

## ケーススタディ：具体的な分析プロセス

### 事例：山田家（夫45歳・妻42歳・子供2人）

**基本情報**
- 夫年収：800万円（手取り約620万円）
- 妻パート収入：150万円（手取り約130万円）
- 資産：預貯金500万円、投資信託200万円、自宅（評価3,000万円）
- 負債：住宅ローン残2,000万円

**分析プロセス**

1. **収支分析**
   - 年間総収入：750万円
   - 年間支出：680万円
   - 年間貯蓄：70万円
   - 貯蓄率：9.3%（改善の余地あり）

2. **支出構造の詳細分析**
   - 固定費：380万円（55.7%）
   - 変動費：300万円（44.3%）
   - 改善可能な項目：食費（年間180万円）、通信費（年間36万円）

3. **資産負債分析**
   - 総資産：3,700万円
   - 総負債：2,000万円
   - 純資産：1,700万円
   - 純資産率：45.9%（改善必要）

4. **ライフプーン分析**
   - 子供の教育資金：約2,000万円必要
   - 老後資産目標：6,000万円
   - 現状のままでは不足額：約3,000万円

5. **改善提案**
   - 支出の見直し：年間30万円の改善余地
   - 投資の開始：毎月5万円の積立投資
   - 妻の正社員化：年収200万円の増収見込み

- メリット：証券口座連携、投資信託管理

- 料金：無料版あり、有料版月額648円

## FP1級試験での出題傾向

### 過去問分析

**出題パターン**：
1. 財務諸表作成問題（約35%）
2. 分析指標の計算問題（約25%）
3. 改善提案の事例問題（约30%）
4. ツール活用に関する問題（約10%）

**頻出計算問題**：
- 貯蓄率の計算
- 返済負担率の計算
- リタイアメント必要資産額の試算
- 教育資金の積立シミュレーション

**注意すべきポイント**：
- 小数点以下の処理方法（切り捨て・四捨五入）
- 年単位か月単位かの明確化
- 税金・社会保険料の控除後での計算

### 効率的な学習方法

1. **計算問題の徹底反復**
   - 公式の完全暗記
   - タイムアタックでの練習
   - 典型的な数字の暗記（貯蓄率20%など）

2. **事例問題の型理解**
   - 改善提案の型を学ぶ
   - 優先順位付けの方法
   - 数字根拠の提示方法

3. **最新ツールの把握**
   - 人気アプリの特徴比較
   - ダウンロード数と評価のチェック
   - 最新機能の把握

## まとめ：財務分析はFPの基礎スキル

財務分析はFPの基礎でありながら、最も重要なスキルです。正確な分析なくして的確な提案はできません。

**重要なポイントの再確認**：
1. 3つの財務諸表を正確に作成・分析できる
2. 分析指標を計算し、意味を解釈できる
3. 改善提案を具体的かつ実行可能にできる
4. デジタルツールを適切に活用できる
