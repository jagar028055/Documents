# 【FP1級取得への道 第4回】資産形成編：運用の実践～ポートフォリオ構築の技術

## はじめに：運用は資産形成の加速装置

FP1級取得に向けた連載第4回。今回は「資産形成編」として、資産運用の実践について詳しく解説します。

なぜ運用が重要なのか？単純な貯蓄だけでは、インフレーションや長寿命化に対応しきれません。お金に働いてもらうことで、複利の効果を最大限に活かし、資産形成を加速させることができます。

FPとして、お客様の資産運用を支援するためには、市場の動向を把握し、適切なポートフォリオを構築し、継続的な見直しができる専門知識が求められます。

## 資産運用の基本：リターンとリスクの関係

### リスク・リターンの基本法則

**リスクとリターンの相関関係**
```
高いリターン ←→ 高いリスク
低いリターン ←→ 低いリスク
```

**主要な資産クラスの特性**

| 資産クラス | 期待リターン | リスクレベル | 特徴 |
|---|---|---|---|
| 現金・預貯金 | 0.01～0.5% | 最低 | 安全性重視、インフレに弱い |
| 国債 | 0.1～1.5% | 低 | 元本確実、安定した利回り |
| 社債 | 0.5～3.0% | 中低 | 信用リスクあり |
| 投資信託 | 3～8% | 中 | 分散投資、専門家運用 |
| 株式 | 5～15% | 高 | 価格変動大、成長性期待 |
| 不動産 | 3～10% | 中高 | インフレヘッジ、流動性低 |

### モダンポートフォリオ理論

**基本コンセプト**
- 複数の資産を組み合わせることで、リスクを低減できる
- 相関の低い資産の組み合わせが効果的
- 効率的フロンティア上のポートフォリオが最適

**分散投資の3原則**
1. **資産クラスの分散**
   - 株式、債券、不動産、現金
   - 国内資産と海外資産のバランス

2. **地域の分散**
   - 先進国市場と新興国市場
   - 通貨の分散投資

3. **時間の分散**
   - ドル・コスト平均法の活用
   - 定期的な積立投資

## NISAとiDeCo：税制優遇制度の活用

### NISA（少額投資非課税制度）の仕組み

**2024年改正NISAのポイント**

```
新しいNISA制度（2024年～）
├── つみたて投資枠：年間120万円
├── 成長投資枠：年間240万円
├── 非課税保有期間：無期限
├── 生涯非課税限度額：1,800万円
└── 売却後の非課税枠の再利用：可能
```

**つみたて投資枠の対象商品**
- 長期の積立・分散投資に適した投資信託
- 手数料が一定水準以下
- 信託期間が20年以上

**成長投資枠の対象商品**
- 上場株式
- 投資信託（一部除く）
- ETF
- REIT

### iDeCo（個人型確定拠出年金）の活用

**iDeCoの特徴**
```
iDeCoのメリット
├── 掛金全額が所得控除（小規模企業共済等掛金控除）
├── 運用益が全額非課税
├── 受給時に公的年金等控除
└── 原則60歳まで引き出せない（強制積立）
```

**掛金限度額（月額）**
- 自営業：68,000円
- 会社員：第2号被保険者
  - 企業年金あり：12,000～23,000円
  - 企業年金なし：23,000円
- 公務員・第2号被保険者：12,000円
- 主婦等：23,000円

### FP1級で求められる制度活用技術

1. **NISAとiDeCoの組み合わせ戦略**
   - iDeCo：節税効果を最大限に活用
   - NISA：中長期の資産形成
   - 特定口座：短期的な運用

2. **ライフステージに応じた活用法**
   - 若年期：iDeCo優先で節税効果最大化
   - 中年期：NISAとiDeCoのバランス
   - 高年期：NISA重点で受取時の税制優遇

3. **非課税枠の最適化**
   - 生涯非課税限度額の計画的使用
   - 売却・再投資のタイミング

## 投資信託：分散投資の王道

### 投資信託の基本

**投資信託の仕組み**
```
多数の投資家から資金を集める
↓
専門家が資産を運用・管理
↓
運用成果を投資割合に応じて分配
```

**投資信託の分類**

1. **投資対象による分類**
   - 株式投資信託：株式中心
   - 債券投資信託：債券中心
   - バランス型：株式・債券の組み合わせ
   - REIT投資信託：不動産投資信託

2. **運用方法による分類**
   - パッシブ（インデックス）：指数連動
   - アクティブ：積極的な運用
   - ソーシャルレンティング：ESG考慮

### 投資信託選定のポイント

**FP1級で求められる分析視点**

1. **パフォーマンス評価**
   - 3年・5年・10年の騰落率
   - ベンチマークとの比較
   - シャープレシオ（リスク調整後リターン）

2. **コスト分析**
   ```
   購入時手数料：0～3.03%（ノーロードが主流）
   信託報酬：0.1～2.0%（年率）
   信託財産留保額：0～0.5%
   監査報酬等：0.001～0.005%
   ```

3. **運用体制の確認**
   - ファンドマネージャーの経験
   - 運用プロセスの明確性
   - ディスクロージャーの質

## 株式投資：成長性を追求する投資

### 株式投資の基本

**株価の決定要因**
```
株価 = PER × EPS
PER：株価収益率（市場の期待）
EPS：一株当たり利益（企業の実力）
```

**株式投資の収益源**
1. **キャピタルゲイン**
   - 株価上昇による売買差益
   - 長期的な成長期待

2. **インカムゲイン**
   - 配当金
   - 株主優待

### 株式分析の手法

**ファンダメンタルズ分析**

1. **財務分析**
   - PER、PBR、ROE、ROA
   - 自己資本比率、流動比率
   - 売上高利益率、営業利益率

2. **業績分析**
   - 売上高の成長性
   - 利益率の推移
   - キャッシュフローの健全性

3. **業界分析**
   - 市場の成長性
   - 競争環境
   - 規制の影響

**テクニカル分析**
- ローソク足のパターン
- 移動平均線
- RSI、MACDなどの指標

## 債券投資：安定性を重視する投資

### 債券の基本

**債券の仕組み**
```
投資家が資金を提供（債権者）
↓
国・企業が資金を借入（債務者）
↓
利息の支払い＋元本の返済
```

**債券の種類**
1. **国債**
   - 最も安全
   - 金利は低い

2. **地方債**
   - 地方公共団体が発行
   - 国債より少しリスクが高い

3. **社債**
   - 企業が発行
   - 信用リスクあり
   - 金利が高い

4. **外国債券**
   - 為替リスクあり
   - 高金利期待

### 債券投資のポイント

**格付けの確認**
- S&P、ムー�ーズ、格付投資情報センター
- AAA～BBBまでが投資適格債
- BB以下が投機的債券（ジャンク債）

**デュレーションの理解**
```
デュレーション = 金利変動に対する価格感応度
数値が大きいほど金利変動の影響が大きい
```

## 積立投資：複利の力を活かす

### ドル・コスト平均法

**効果**
- 購入単価の平準化
- 時間分散によるリスク低減
- 心理的な負担軽減

**実践例**
```
毎月1万円積立
月次価格：1万円、1万2千円、8千円、1万1千円...
購入口数：1口、0.83口、1.25口、0.91口...
平均購入単価の引き下げ効果
```

### 複利の計算

**複利の公式**
```
将来価値 = 現在価値 × (1 + 利率)^期間
```

**具体例**
```
毎月3万円を年利5%で20年間運用
最終積立額：720万円
元利合計：1,237万円
利息部分：517万円（約72%の利益）
```
