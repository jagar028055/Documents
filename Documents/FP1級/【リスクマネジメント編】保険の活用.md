# 【FP1級取得への道 第3回】リスクマネジメント編：保険の活用～適切な保障設計の技術

## はじめに：保険はリスク管理の要

FP1級取得に向けた連載第3回。今回は「リスクマネジメント編」として、保険の活用について詳しく解説します。

なぜ保険が重要なのか？人生には予期せぬリスクが常に存在します。病気、怪我、死亡、火災、自然災害——これらのリスクが実際に発生したとき、経済的な打撃を最小限に抑えるのが保険の役割です。

FPとしてお客様のリスク管理を支援するためには、単なる保険商品の知識だけでなく、リスクの本質を理解し、適切な保障設計ができる能力が求められます。

## リスク管理の基本：リスクの分類と対策

### リスクの種類と特徴

**人生の三大リスク**
1. **長生きのリスク（長寿リスク）**
   - 老後資金が不足するリスク
   - 対策：個人年金保険、投資信託、iDeCo・NISA

2. **病気やケガのリスク（健康リスク）**
   - 医療費や長期療養による収入減少
   - 対策：医療保険、がん保険、就業不能保険

3. **死亡のリスク**
   - 早期死亡による遺族の生活困窮
   - 対策：生命保険（定期保険、終身保険）

### リスク管理の四大手法

1. **回避（Risk Avoidance）**
   - 危険な行動を避ける
   - 例：禁煙、運転を控える

2. **軽減（Risk Reduction）**
   - リスク発生率を下げる
   - 例：健康診断、定期的な健康維持

3. **移転（Risk Transfer）**
   - リスクを第三者に移す
   - 例：保険への加入

4. **保有（Risk Retention）**
   - 自己負担で対応
   - 例：少額の治療費は自己資金で

## 保険の基本：保障が必要な理由

### 公的保障の限界

**日本の公的医療保障**
- 健康保険：治療費の70%をカバー（自己負担30%）
- 高額療養費制度：月々の自己負担を上限額に
- 介護保険：要介護状態になった場合の支援

**公的保障だけでは不十分な理由**：
1. 自己負担額が依然として大きい
2. 先進医療や自由診療は対象外
3. 収入減少には対応できない
4. 生活の質（QOL）の維持は難しい

### 保険が必要な具体的場面

**医療費の実態（2024年データ）**
- 平均入院日数：約28日
- 入院1回あたりの平均医療費：約40万円
- 自己負担額（3割）：約12万円
- 高額療養費適用後：約5～8万円

**がん治療の現実**
- がん保険での平均給付金額：約100万円
- 先進医療の平均自己負担：約200万円
- 治療期間の平均：3～5年

## 医療保険：病気・ケガへの備え

### 医療保険の基本構造

**主な保障内容**
1. **入院給付金**
   - 入院日数に応じて支給
   - 最近は「入院日数無制限」が主流

2. **手術給付金**
   - 手術の種類に応じて支給
   - 手術の回数制限あり

3. **通院給付金**
   - 入院前後の通院を保障
   - 最近は通院特約が充実

4. **特定疾病保障**
   - がん、急性心筋梗塞、脳卒中の三大疾病
   - 一時金で支給される場合が多い

### 医療保険選定のポイント

**FP1級で求められる分析能力**：

1. **給付金日額の適正評価**
   ```
   給付金日額の目安 = 現在の月収 ÷ 30
   現実的な上限：月収の60%程度
   ```

2. **免責期間の理解**
   - 入院保障：待機期間あり（通常3ヶ月）
   - がん保障：無給付期間（通常90日）
   - 給付制限：責任開始期からの制限

3. **更新タイプの比較**
   ```
   終身更新型：保険料変動、一生保障
   - 定期更新型：保険料固定、期間限定
   - 終身保険型：保険料固定、一生保障
   ```

### 実務での相談事例

**ケースA：30歳・既婚・子どもなし**
- 月収：30万円
- 現状：会社員で社会保険完備
- 提案：月5,000円の就業不能特約付医療保険

**ケースB：40歳・自営業・子ども2人**
- 月収：50万円
- 現状：国民健康保険、会社の保険なし
- 提案：月15,000円の医療保険＋就業不能保険

## がん保険：現代の必須保障

### がんの現実（2024年データ）

**がんの統計**
- 生涯でがんにかかる確率：男性65%、女性51%
- 年間新規罹患数：約100万人
- 5年生存率：全体で約66%（部位により大きく異なる）

**がん治療費の内訳**
- 初期治療費：平均300万円
- 後期治療費：平均200万円
- 先進医療：100～500万円
- 代替医療：50～200万円

### がん保険の選定基準

**保障内容の比較ポイント**：

1. **診断一時金**
   - 初回診断時に一括給付
   - 金額：100万～500万円が主流
   - 1回限りか複数回か

2. **入院給付金**
   - 日額：5,000円～2万円
   - 保障期間：5年・10年・60歳・終身
   - 上限ありか無制限か

3. **通院給付金**
   - 入院前後の日数
   - 放射線治療などの特例

4. **先進医療保障**
   - 先進医療技術の対象
   - 上限額：200万～1,000万円

**FP1級で求められる分析**：
- がん治療の最新動向の把握
- 遺伝的要因の考慮
- 家族歴の活用方法

## 就業不能（所得補償）保険：収入の安心

### 就業不能の定義と種類

**就業不能の定義**
``業務外の病気やケガで、医師の指示により入院または自宅療養が必要で、
仕事が全くできない状態``

**種類別就業不能保険**：

1. **所得補償保険**
   - 収入に応じた保障
   - 月給の50～80%が目安
   - 就業不能期間中の全期間保障

2. **就業不能特約**
   - 医療保険に付帯
   - 給付金額：5,000円～2万円/日
   - 保険期間の制限あり

3. **就業不能年金**
   - 年金形式で支給
   - 保障期間：5年・10年・65歳まで
   - 受取り開始の待機期間あり

### 所得補償保険の設計技術

**保障額の算出方法**：
```
必要保障額 = 月収 × (1 - 家計費比率) × 70%
```

**給付期間の選定基準**：
- 年齢：若いほど長期保障を
- 職業：安定職は短期でもOK
- 家族構成：扶養家族が多いほど長期を

**免責事項の確認**：
- 自己傷害、妊娠・出産
- 職業病、故意による事故
- 麻薬・アルコール関連

## 生命保険：遺族への最後の愛

### 死亡保障の必要性

**遺族が必要とする資金**

1. **葬祭費**
   - 平均費用：約200万円
   - 宗教・地域による差異

2. **生活再建資金**
   - 1～2年分の生活費
   - 住宅ローン返済額

3. **教育資金**
   - 大学進学までの費用
   - 子供1人あたり1,000万円

4. **老後資金**
   - 配偶者の老後資金
   - 公的年金だけでは不足

### 保険種類の選定技術

**定期保険**
- メリット：保険料が安い、必要額だけ保障
- デメリット：保険期間の限定、更新時の保険料急増
- 適した人：若年層、住宅ローン返済中

**終身保険**
- メリット：一生保障、解約返戻金あり
- デメリット：保険料が高い
- 適した人：中高年、資産形成目的

**収入保障保険**
- メリット：保険料が安い、遺族が自由に使える
- デメリット：保障額が徐々に減少
- 適した人：子供のいない夫婦、高齢者

### FP1級レベルの設計技術

1. **複数保険の組み合わせ**
   - 住宅ローン：定期保険
   - 遺族生活：収入保障保険
   - 資産形成：終身保険

2. **保険料配分の最適化**
   - 若年期：死亡保障重視
   - 中年期：医療＋死亡保障
   - 老年期：資産形成＋医療保障

3. **見直し時期の判断**
   - 結婚・出産・子の独立
   - 住宅ローン完済
   - 定年退職

## 保険ポートフォリオの最適化

### ライフステージ別保険設計

**20代・独身**
```
死亡保障：少額（500万円程度）
医療保障：会社の保険で十分
がん保障：医療保険のがん特約で対応
就業不能：社会保険でカバー
```

**30代・夫婦・幼児**
```
死亡保障：夫2,000万円・妻1,000万円
医療保障：家族プランで充実
がん保障：夫婦ともに検討
就業不能：主たる稼ぎ手に重点
```

**40代・夫婦・学齢期**
```
死亡保障：夫3,000万円・妻1,500万円
医療保障：先進医療特約を追加
がん保障：先進医療対応のものを
就業不能：夫婦ともに必要
```

**50代・夫婦・子独立**
```
死亡保障：徐々に減額
医療保障：終身保障タイプへ
がん保障：終身保障タイプへ
就業不能：必要に応じて継続
```

### 保険料の適正水準

**保険料の目安**
```
世帯年収に対する保険料比率
- 20～30代：5%以内
- 40～50代：8%以内
- 60代以上：6%以内
```

**過剰保障の見極め**
- 収入に見合わない高額保障
- 重複する保障内容
- 不要な特約の付帯

**FP1級で求められる判断能力**：
- 適正保険料の算出
- 過剰・不足保障の是正
- 無理のないプラン提案

## 保険見直しのタイミングと方法

### 見直しが必要なライフイベント

1. **結婚・離婚**
   - 扶養家族の変化
   - 必要保障額の再計算

2. **出産**
   - 子供の教育資金
   - 遺族生活資金の増加

3. **住宅購入**
   - 住宅ローン返済中の死亡保障
   - 団体信用生命保険との調整

4. **転職・独立**
   - 会社保険の変更
   - 社会保険の種別変更

5. **子供の独立**
   - 保障額の減額
   - 保険料の軽減

### 見直しの具体的ステップ

**STEP1：現状の確認**
- 加入中の保険内容の整理
- 保険料と保障内容の比較
- 必要保障額の再計算

**STEP2：比較検討**
- 複数のプランを比較
- 保険料と保障内容のバランス
- 将来の見直しやすさ

**STEP3：実行**
- 不要保険の解約
- 新規保険の加入
- 特約の追加・削除

## FP1級試験での出題傾向

### 過去問分析

**出題パターン**：
1. 保険商品の特性問題（約30%）
2. 保障設計の事例問題（约35%）
3. 保険料計算問題（约20%）
4. 法規・倫理関連問題（约15%）

**頻出計算問題**：
- 必要保障額の計算
- 保険料の比較計算
- 収入保障保険の年金計算
- 保険期間の算出

**注意すべきポイント**：
- 保険料は税引き前か後か
- 保障額は現在価値か将来価値か
- 家族構成の変化を考慮

### 効率的な学習方法

1. **保険商品の体系的整理**
   - 分類別の特徴まとめ
   - 比較表の作成
   - 用語の完全理解

2. **事例問題のパターン学習**
   - 典型的なライフステージの設計
   - 選定理由の明確化
   - 複数プランの比較検討

3. **最新動向の把握**
   - 新商品の特徴理解
   - 保険法の改正点
   - 社会保障制度の変更
