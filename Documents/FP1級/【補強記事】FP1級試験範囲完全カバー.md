## 1. リタイアメントプランニング：公的年金の完全解説

### 公的年金制度の3階建て構造

**第1階層：基礎年金（国民年金）**
- 被保険者：全国民
- 保険料：月16,520円（2024年度）
- 満額年金：月68,000円（2024年度価格）

**第2階層：厚生年金・共済年金**
- 被保険者：会社員・公務員
- 保険料：標準報酬月額×18.3%（会社折半）
- 年金額：平均的男性で月約15万円

**第3階層：企業年金・個人年金**
- 確定拠出年金（DC）
- 確定給付企業年金（DB）
- 个人型確定拠出年金（iDeCo）

### 年金給付の詳細計算

**老齢基礎年金の計算**
```
満額受給資格期間：40年（480月）
年金額 = 68,000円 × （加入月数/480月）
```

**老齢厚生年金の計算**
```
年金額 = 報酬比例部分 + 加給年金額

報酬比例部分 = 平均標準報酬月額 × 7.125/1000 × 加入月数
             + 平均標準報酬額 × 5.481/1000 × 加入月数
```

**在職老齢年金の仕組み**
```
総収入額 = 賃金＋年金
65歳未満：28万円超で年金一部停止
65歳以上：46万円超で年金一部停止
```

### FP1級で必須の年金知識

1. **年金繰上げ・繰下げ**
   - 繰上げ：65歳～60歳、減額率0.5%/月
   - 繰下げ：65歳～70歳、増額率0.7%/月

2. **年金生活者支援給付金**
   - 所得一定以下の高齢者に支給
   - 基礎年金の最大約1.5万円を上乗せ

3. **障害年金**
   - 障害基礎年金：1級・2級
   - 障害厚生年金：1級～3級
   - 障害手当金：一時金

## 2. 法律関連：FP1級で必須の法的知識

### 民法（親族法・相続法）

**婚姻の要件**
- 実質的要件：当事者の意思
- 形式的要件：届出

**夫婦財産制**
- 法定財産制：別産制と共有制
- 契約財産制：婚前契約等

**親権**
-身上監護権と財産管理権
- 未成年後見との関係

### 成年後見制度

**法定後見**
1. 後見：判断能力全くない
2. 保佐：判断能力特に著しく不十分
3. 補助：判断能力が不十分

**任意後見**
- 本人の判断能力があるうちに契約
- 公正証書で作成
- 登記が必要

**後見人の選任**
- 複数選任も可能
- 法人の後見人も可
- 後見監督人制度

### 信託法

**信託の基本原則**
- 信託財産の独立性
- 受益者平等の原則
- 信託財産の分別管理

### 消費者契約法

**消費者契約の取消権**
- 不実告知：誤認させた場合
- 断定的判断の提供：困惑させた場合
- 不利益事実の不告知：故意に告げなかった場合

取消期間：追認可能時から5年、契約時から20年

## 3. FP技能士制度の詳細

### 資格の種類と区分

**FP技能士**
- 1級：国家資格、高度な専門知識
- 2級：準国家資格、基本的知識
- 3級：入門レベル

**AFP・CFP®**
- AFP：Afiliated Financial Planner
- CFP®：Certified Financial Planner

### 試験制度の詳細

**学科試験**
- 実施回数：年2回（1月・6月）
- 試験時間：2時間30分
- 問題数：60問（5肢選択）
- 合格基準：70%以上

**実技試験**
- 実施回数：年2回（学科後）
- 試験時間：2時間30分
- 問題形式：記述式
- 評価基準：分析力・提案力・表現力・倫理観

### 資格更新制度

**継続教育（CE）**
- 3年ごとに15単位以上取得
- 認定研修：1研修0.5単位
- 通信教育：1講座0.2単位
- その他：FP協会主催研修等

**資格更新手続き**
- 更新期間：3年ごと
- 更新料：6,600円
- CE単位の証明が必要
